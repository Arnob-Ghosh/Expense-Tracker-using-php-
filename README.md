# Expense-Tracker-using-PHP-
a web app that can track your day-to-day expenses  

index.php: The index is the dashboard of the system. It has a Navigation bar that contains all the links that we need and in the body, we can see a table that contains all the expenses we can also edit or delete expenses from here
Add Expense: we can see add an expense in the navigation bar in second place which allows users to add cost and prevent from adding future dates
Add Category: this page can add new categories to the database
View Category: the view category page shows all the classes in the database and can let the user update or delete any category
Expense between dates: The expense between dates  page takes two dates as input and shows the costs that were added between those dates
